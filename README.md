ClassAttend is a role-based attendance management system.
It helps colleges manage attendance digitally using HOD, Teacher,
and Student roles.


## Tech Stack

### Frontend
- react.js javascript  tailwind css redux
- javascript
- tailwind css 
-  redux

### Backend
- Node.js
- Express.js
- MySQL


## Features

### HOD
- Create and manage teachers
- Create classes and subjects
- Assign teachers to classes
- Create and mange students
- and analysis student class subject scores 

### Teacher
- View assigned classes
- Mark student attendance
- View attendance records
- analysis student score

### Student
- View attendance history
- view attendance score
